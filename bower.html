The Me-Guide to Bower

What is Bower and Why

Web sites are made of lots of things (frameworks, libraries, assets...). Lets say that we want to use bootstrap, jquery, and less in our website. Instead of going to every of those websites and download them directly, we can do it with Bower

Bower is a package manager. It fetches all the packages you need, it downloads and installs them for you.

How to Bower

Install Bower

$ npm install -g bower

(It requires Node, Npm and Git)

Check version

$ bower --v

Search Packages

Now you can already look for all those frameworks, libraries and assets you want to have in your application. Probably these are some of the ones you want: Bootstrap, Jquery, Fontawesome, Modernizr...

For the full list of packages go to this link

How does your folder look like?

So far you have not installed anything so apart from the files and folders you created there is nothing there. So lets install some things.

Install the Packages!

Once you have decided which packages you need, you can easily install them. Usually using the package name:

$ bower install "package-name"

$ bower install jquery

How does your folder look like?

You now have a folder called "bower_components". Inside it one folder per installed package

Save your Packages

If you want you can save your packages so that others can use them and install them easily

$ bower init

This will create a file called bower.json. This file is where Bower keeps track of your packages

It will ask you a set of questions starting with the name you want to give your package. Press enter to all question if you want to choose the default options. For further information follow this link

From this moment on, you should either save any new intalled package in the json:

$ bower install jquery --save

or run bower init again to refresh the json file

$ bower init

How does your folder look like?

You now have the folder called "bower_components", and a file called "bower.json". This is the file where Bower keeps track of your chosen packages
